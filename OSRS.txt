# OSRS RuneLite Mobile Platform Summary

## Key Facts
- **No official RuneLite mobile app** - only desktop client works
- **Touchscreen support is poor** - camera rotation requires middle mouse button
- **Best for:** Skilling, AFK activities, questing, basic PvM
- **Avoid for:** PKing, tick manipulation, high-intensity PvM

## Platform Options

### Steam Deck OLED - $549-$649 ⭐ **RECOMMENDED**
- **Pros:** Best overall experience, great battery, refined software
- **Cons:** 720p resolution, gamepad-only controls
- **Setup:** Desktop mode → Download RuneLite AppImage → Add to Steam

### ASUS ROG Ally X - $799
- **Pros:** Most powerful, 1080p, best specs, longest battery
- **Cons:** Expensive, OpenGL crashes with 117HD plugin
- **Issues:** GPU plugin problems, driver compatibility

### Lenovo Legion Go - $699
- **Pros:** Detachable controllers with mouse mode, large 8.8" screen
- **Cons:** Expensive, shorter battery life
- **Best for:** Users wanting mouse precision

### Windows Tablets (Surface) - $300-$800
- **Pros:** Cheapest option, full Windows
- **Cons:** **Requires external mouse**, poor touch experience
- **Only viable with:** Bluetooth mouse + keyboard

## Bottom Line
- **Best Value:** Steam Deck OLED ($549) - perfect for casual OSRS
- **Most Capable:** Legion Go ($699) - handles harder content better
- **Budget:** Used Surface Pro + mouse ($200-400) - skilling only
- **Skip:** Pure tablets, ROG Ally (plugin issues), anything Android/iOS
